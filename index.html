Про меня

Привет!

Привет! Меня зовут Артём, я web-разработчик, специализирующийся на создании современных и удобных интерфейсов. Пишу чистый, оптимизированный код и разрабатываю функциональные, производительные решения для пользователей.

Владею HTML, CSS и JavaScript. Использую Vue 3 с Composition API, Pinia для управления состоянием, Vue Router для маршрутизации. Разрабатываю адаптивные интерфейсы с упором на скорость и удобство.

В работе применяю Vite для быстрой сборки, Git для контроля версий, ESLint + Prettier для чистоты кода. Работаю с Axios / Fetch API для взаимодействия с сервером.

Постоянно совершенствую свои навыки и слежу за новыми технологиями. Если вам нужен ответственный web-разработчик, готовый к продуктивной работе — буду рад присоединиться к вашей команде!

Навыки

Что я использую в работе
HTML5
HTML5 — основа веб-разработки. Позволяет создавать семантические страницы, поддерживает мультимедиа, анимации и API для браузера.
CSS3 & SCSS
CSS3 используется для стилизации сайтов, а SCSS расширяет его возможностями переменных, миксинов и вложенности, что делает код более удобным для поддержки.
JavaScript (ES6+)
Современный JavaScript (ES6+) включает стрелочные функции, модули, асинхронный код и классы, что делает разработку удобнее и мощнее.
Vue.js
Vue.js используется для создания интерактивных веб-интерфейсов, упрощая управление состоянием, рендеринг и разработку модульных компонентов.
Vite
Vite – это быстрый инструмент сборки для Vue и других фреймворков, который обеспечивает мгновенную перезагрузку и быструю компиляцию.
Pinia
Pinia – это удобный и гибкий менеджер состояния для Vue.js, который заменяет Vuex. Позволяет легко управлять данными в приложении.
Vue Router
Vue Router – это официальная библиотека маршрутизации для Vue.js. Она позволяет создавать динамические страницы и управлять навигацией в приложении.
Git & GitHub
Git – это система контроля версий, а GitHub – платформа для совместной работы над кодом, хранения репозиториев и CI/CD.
Webpack & Gulp
Webpack – это мощный бандлер для JavaScript, а Gulp – инструмент автоматизации задач, таких как сборка CSS, минификация и копирование файлов.
PHP
PHP – это язык серверного программирования, используемый для обработки данных, генерации динамических страниц и взаимодействия с базами данных.
